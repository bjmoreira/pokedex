## Instruções para instalação

Execute os seguintes comandos na sequencia:

composer install

php artisan migrate

php artisan db:seed

npm install

npm run dev


Setar a variavel no .env, para a porta que está em uso pelo artisan.

APP_URL=http://localhost:8000
